# (WIP) React Component Generate

Generate a structure for react components.

```
ComponentName
    - ComponentName.tsx
    - index.ts
    - helpers
        - function.ts
    - __test__
        - ComponentName.test.ts
        - function.test.ts
    - __spec__
        - ComponentName.stories.tsx
```

## Usage

```
react-component-gen Accordion
```
